🤖 AI Nav - AI工具导航 API

为 AI Agents 提供的 AI 工具发现和推荐接口

📡 API 端点

GET /api/tools

获取所有 AI 工具列表

GET /api/tools?category=AI对话&freeOnly=true
GET /api/tools/ai-friendly

AI 友好的简化响应格式

GET /api/tools/ai-friendly?q=图像生成&limit=5
GET /api/tools/search?q=<关键词>

搜索 AI 工具

GET /api/tools/search?q=代码
GET /api/tools/:id

获取工具详情

GET /api/tools/chatgpt
GET /api/categories

获取所有分类

🤖 MCP Tools

GET /api/mcp

获取 MCP 服务信息和可用工具

POST /api/mcp

调用 MCP 工具

curl -X POST /api/mcp \
  -H "Content-Type: application/json" \
  -d '{"name":"search_ai_tools","arguments":{"query":"图像生成"}}'

🛠️ 可用 MCP 工具

  • search_ai_tools - 搜索 AI 工具
  • get_tool_details - 获取工具详情
  • list_categories - 列出所有分类
  • get_by_category - 按分类获取工具
  • get_recommendations - 获取推荐

📊 已收录工具

共收录 12 个精选 AI 工具，覆盖：

  • AI 对话 (ChatGPT, Claude)
  • AI 图像 (Midjourney, DALL-E, Stable Diffusion)
  • AI 编程 (GitHub Copilot, Cursor)
  • AI 视频 (Runway)
  • AI 音频 (ElevenLabs)
  • AI 自动化 (Zapier, Notion AI)
  • AI 搜索 (Perplexity)

🔧 测试命令

# 获取所有工具
curl http://localhost:3002/api/tools

# AI 友好查询
curl "http://localhost:3002/api/tools/ai-friendly?q=图像生成"

# 搜索工具
curl "http://localhost:3002/api/tools/search?q=代码"

# MCP 调用
curl -X POST http://localhost:3002/api/mcp \
  -H "Content-Type: application/json" \
  -d '{"name":"get_recommendations","arguments":{"use_case":"编程","free_only":true}}'
